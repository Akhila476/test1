- Navigate to the document via [Search](https://pages.github.ibm.com/Global-EJS/gejs-user-manual/docs/DocumentSearch/DocumentSearch.html)  or for Indexer user access by expanding the Global_EJS folder to Payroll_US folder and the Workbasket sub-folders.
- Users may view documents via Search Results or by selecting the Workbasket folder (based on user permissions).
- Right click on the document, menu action will be opened.
- Click on “Print” 
- A print window is displayed.

    ![image](https://media.github.ibm.com/user/369573/files/e48c3600-cbc6-11ec-8324-dd64ea8dabe3)

- Choose the “Print Original size” checkbox, if not already checked.
- Click on “Print”.

    ![image](https://media.github.ibm.com/user/369573/files/f66dd900-cbc6-11ec-9e51-f3bb2c4a9973)

- Click on Print button to print the document.

    ![image](https://media.github.ibm.com/user/369573/files/100f2080-cbc7-11ec-9476-ce6d4c24f6b6)

- **Note:** The print option for the “Txt” files:-
  + A print option is not enabled on the document of type "Text" files.
  + To print documents of type “Text”, an internet browser requires the support of the Java applet but due to increased security in the latest internet browser client, Java applet is not supported. 

    ![image](https://media.github.ibm.com/user/369573/files/39c84780-cbc7-11ec-9bdd-7ed0366564fa)

- **Resolution** - Instead of clicking on the print menu option for “Text” documents click the “Preview” menu option. 
 
    ![image](https://media.github.ibm.com/user/369573/files/0803b080-cbc8-11ec-9b1e-68f46b8391ab)

- The text file will open in the content navigator viewer and the user can print a document.

    ![image](https://media.github.ibm.com/user/369573/files/18b42680-cbc8-11ec-8118-8a074e18a147)

- Click on the Print icon.

    ![image](https://media.github.ibm.com/user/369573/files/36818b80-cbc8-11ec-8020-cee1dc7ed057)

- Click on ‘Prepare File’ button

    ![image](https://media.github.ibm.com/user/369573/files/48fbc500-cbc8-11ec-829f-2967805d282d)

- Click on’Print’ button

    ![image](https://media.github.ibm.com/user/369573/files/63ce3980-cbc8-11ec-890e-5cea318a68e4)


- Once click on ‘Print button’ below popup message will be populated on the screen.
- User should provide the name of the document and then click on save button, now the document will be saved in selected location with provided file name

    ![image](https://media.github.ibm.com/user/369573/files/83656200-cbc8-11ec-99ee-012b62949224)
